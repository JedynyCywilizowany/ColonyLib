This is a library mod, does nothing on its own.
Made for my other mods, though others may also use it in thier own.

If you want to see my other mods, go there:
https://steamcommunity.com/profiles/76561199099032737/myworkshopfiles/?appid=1281930&sort=score&browsefilter=myfiles&view=imagewall 


Features:

- ColonyPacketType, a ModType to help with managing mod packets

- New base types for content, prefixed with "Colony":
>> optionally change the default texture path to my own convention
>> ColonyItem lets body armor draw extra textures over legs, without replacing them like robes do. This effect is hard-coded into some vanilla vanity items, but there is no official mod support for it.

- ColonyUtils:
>> RevertibleModify = modify an array (usually an ID set), then automatically revert the change when mods unload
>> SerializeBools/DeserializeBools = efficiently serialize/deserialize arrays and lists of booleans
>> AddIfNotEmpty/AddIfNotDefault = only add an entry into a TagCompound if it's not empty/default, without having to check for it manually
>> GetMapColorTile/GetMapColorWall = get the color of a tile/wall type on the map
>> DummyItems/DummyNPCs = easy and efficient access to reusable instances of items/NPCs by type
>> AveragedInt = cast a float/double to int, randomly rounding it up or down so that the average stays the same
>> UpdateTooltip = a single function for adding, modifying or removing a tooltip
>> + a few more things that I didn't bother to list here

- Debug Mode, made to help with debugging mods that depend on it
>> Optionally log all uses of ColonyPacket in the chat
>> Allows adding WorldGen Markers; permanent map icons to help locating own structures and other points of interest
